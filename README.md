# 🔥 Calculator

A modern and responsive calculator built using **HTML, CSS, and JavaScript** with advanced features like dark/light mode, keyboard support, and calculation history.

---

## 🚀 Features

- ➕ Addition, ➖ Subtraction, ✖ Multiplication, ➗ Division  
- 🌙 Dark / ☀ Light Theme Toggle  
- ⌨ Keyboard Support  
- 🕘 Calculation History  
- ❌ Clear & Delete Functions  
- 📱 Responsive Design for Mobile & Desktop  

---

## 🛠 Tech Stack

- HTML5  
- CSS3  
- JavaScript (Vanilla JS)
